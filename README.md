# MeanDemo

MEAN (Mongodb Express.js Angular.js Node.js) stack RESTful CRUD (using Angular `$resource`) demo using free [MongoLab](www.mongolab.com) account. Enjoy!

# Prerequisites

1. [Node.js & NPM](https://nodejs.org)

# Installation

```
$ npm install
$ node index.js
```
